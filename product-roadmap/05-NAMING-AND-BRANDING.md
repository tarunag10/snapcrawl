# 05 - Naming and Branding

Your tool needs a memorable, searchable name. "set up configuration for screenshots" is a description, not a brand.

---

## Name Criteria

A good dev tool name is:
- **Short** (1-2 words, <12 characters)
- **Available** on npm, GitHub, and as a domain
- **Descriptive** enough to hint at what it does
- **Unique** enough to search for

## Recommended: `snapcrawl`

**Why:**
- Combines "snapshot" and "crawl" - exactly what the tool does
- Short (9 chars)
- Easy to type: `npx snapcrawl`
- Memorable and unique
- Works as a verb: "I snapcrawled my site"

**Taglines:**
- "Auto-crawl any website. Get screenshots + demo videos."
- "Zero-config visual capture for any web app."
- "The free, local alternative to Percy and Chromatic."
- "From URL to screenshot gallery in one command."

## Positioning Statements

### For the npm README (first line)

> Automated website screenshot and demo video generator. Crawls your site, captures every page at multiple viewports, records polished walkthrough videos. Zero config required.

### For Hacker News / Reddit

> Show HN: Snapcrawl - Auto-generate screenshot galleries and demo videos of any website
>
> I built a free CLI tool that crawls your web app, takes screenshots at desktop/mobile/tablet viewports, and records an MP4 demo video with smart interactions (hover, scroll, click exploration). One command, no code needed. Config-driven for power users.
>
> `npx snapcrawl https://your-app.com`

### For GitHub Description

> Crawl any website and auto-generate multi-viewport screenshots + polished demo videos. Free, local, config-driven.

## Logo Ideas

Keep it simple. A camera/lens icon combined with a spider web (crawl) or a grid (viewports).

Colors: Use a bold accent color. Purple (#7C3AED) or Teal (#0D9488) work well for dev tools.

Tools to make it: Figma (free), or ask an AI image generator for "minimal flat icon, camera lens with web grid, purple accent, transparent background."
